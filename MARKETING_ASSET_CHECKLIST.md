# Pantry & Plate Marketing Asset Checklist

Use this checklist before publishing any screenshot, video, reel, ad creative, press image, website image, or social post.

## Store Links And Availability

- [ ] App Store URL is exactly: https://apps.apple.com/us/app/pantry-plate/id6761082174
- [ ] Google Play URL is exactly: https://play.google.com/store/apps/details?id=app.pantryandplate
- [ ] Both store links are live and visible where download CTAs appear.
- [ ] Trial copy says eligible users can start a 7-day free trial.
- [ ] Subscription copy says active subscription required after trial.
- [ ] Copy does not say or imply "free forever."

## Screenshot And Video Safety

- [ ] No `Original line`.
- [ ] No `No nutrition data available yet`.
- [ ] No `Allergens unknown`.
- [ ] No parser labels.
- [ ] No debug labels.
- [ ] No internal QA labels.
- [ ] No `marketing capture`.
- [ ] No reviewer credentials.
- [ ] No support/reviewer email shown inside app UI.
- [ ] No account emails, customer IDs, transaction IDs, API keys, private env values, or service-account files.
- [ ] No black redaction bars. Normal device chrome such as Dynamic Island or home indicator is acceptable when it is part of the capture frame.
- [ ] No placeholder icons.
- [ ] No broken UI, overlapping text, clipped controls, or placeholder failures.
- [ ] No unsafe old media unless it has been re-reviewed after production UI cleanup.

## Claims And Positioning

- [ ] No guaranteed savings claim.
- [ ] No medical, allergy-safety, calorie-tracking, or weight-loss plan claim.
- [ ] No iPad support claim.
- [ ] No cashback, coupon, rewards, discount, deal, or incentive positioning.
- [ ] Optional connected shopping is described as optional and user-started.
- [ ] Connected shopping copy does not imply checkout is handled by Pantry & Plate.
- [ ] AI-assisted recipe and image tools are described as user-started workflows.
- [ ] Imported recipe language does not claim attribution prevents infringement.

## Brand And Product Fit

- [ ] Core message is present where useful: Know what you have. Plan what you'll cook. Shop only what's missing.
- [ ] Product positioning is accurate: pantry tracking, recipe saving and organization, My Cookbook, meal planning, smart grocery lists, household-friendly organization, optional connected shopping, and AI-assisted recipe/image tools.
- [ ] Visuals show clean, public-facing UI only.
- [ ] Captions and overlays are readable on mobile.
- [ ] App Store badge or button links to the live App Store URL.
- [ ] Google Play badge or button links to the live Google Play URL.

## Final Review

- [ ] Asset reviewed at full size.
- [ ] Video reviewed from first frame through final frame.
- [ ] Audio captions/subtitles reviewed if present.
- [ ] Owner approved final public use.
- [ ] File name and destination documented before publishing.
